— Projects —

Bookworm is a book search/recommendation web application built on the Ruby on Rails framework. Visitors can search by title, author or ISBN, and logged in users can create their own reading list by saving book titles to their bookbag.

The GoodReads API was used to retrieve the search results, book details and recommended readings, and the Nokogiri gem was used to parse the XML format returned by the API as well as some screen scraping.

CSS Stache is a web application designed to showcase pure CSS-based designs also built on the Rails framework. I mainly contributed to the back-end of this group project by incorporating the Devise gem for authentication, the Rails Admin gem for administration of the application and adding commenting and tagging features.
